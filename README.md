# Handless

## Dependences

### nodejs 10.5.0 debian install :
```

	sudo apt-get install -y build-essential
    
    curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -
	
	sudo apt-get install -y nodejs

```
### npm 6.1.0
It will be install with nodejs


## Run the project
```
	cd Handless

	npm run serve
```
